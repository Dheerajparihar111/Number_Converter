# 🔢 Number Converter 

A professional **Command Line Interface (CLI) Number Converter** built using **Java** that can convert numbers between different number systems.

This project demonstrates menu-driven programming, built-in Java conversion methods, and user input handling.

---

## 🚀 Features

✔️ Decimal to Binary
✔️ Decimal to Hexadecimal
✔️ Binary to Decimal
✔️ Menu-driven CLI interface
✔️ Clean and simple code
✔️ Uses Java built-in conversion functions

---

## 🛠️ Technologies Used

* ☕ Java
* 💻 Command Line Interface (CLI)
* 🔁 Loops & Switch case
* 📥 Scanner class
* 🔢 Integer conversion methods

```
===== NUMBER CONVERTER =====

1. Decimal to Binary
2. Decimal to Octal
3. Decimal to Hexadecimal
4. Binary to Decimal
5. Octal to Decimal
6. Hexadecimal to Decimal 
7. Exit

```

---

## 📌 Conversion Methods Used

### Decimal → Binary

```
Integer.toBinaryString(number)
```

### Decimal → Hexadecimal

```
Integer.toHexString(number)
```

### Binary → Decimal

```
Integer.parseInt(binary, 2)
```
